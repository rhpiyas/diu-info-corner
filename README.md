# 🎓 DIU Info Corner

A smart and user-friendly application designed to help prospective students get all necessary admission-related information about **Daffodil International University (DIU)** in one place.

The application provides department information, waiver details, seat availability, direct admission eligibility checking, and important admission notifications to make the admission process easier and faster.

---

## 📌 Project Overview

Every year, many students face difficulties finding accurate admission information such as department details, available seats, waiver opportunities, and admission requirements.

**DIU Info Corner** solves this problem by providing a centralized platform where students can easily access all admission-related information.

- ### User mode:
  - Department info
  - Total seat

  - User function

  - View registration data
  - Update user data

  - Update app data

## 🚀 Features

### 👨‍🎓 Student Features

- 📚 **Department Information**
  - View available departments
  - Get department details and descriptions

- 🎫 **Seat Availability**
  - Check available seats for different departments

- 💰 **Waiver Information**
  - View different waiver categories
  - Understand eligibility criteria

- ✅ **Direct Admission Checker**
  - Check admission eligibility based on academic results

- 🔔 **Admission Notifications**
  - Get updates about admission-related announcements

---

### 🔐 Admin Features

- Manage department information
- Update seat availability
- Manage waiver information
- Publish important notifications
- Maintain system data
